# 📌 Project Overview

The **AWS MLOps Heart Disease Prediction System** is an end-to-end Machine Learning deployment project that predicts the risk of heart disease using patient medical parameters.

Unlike traditional ML projects that only focus on model training, this project demonstrates the complete ML lifecycle:

- Dataset Collection
- Dataset Merging
- Data Preprocessing
- Model Training
- Model Evaluation
- Model Selection
- AWS SageMaker Deployment
- FastAPI API Development
- Docker Containerization
- EC2 Hosting
- Nginx Reverse Proxy
- HTTPS Domain Deployment

The complete application is deployed as a production-ready web application.

# 🎯 Problem Statement

Heart disease is one of the major health problems worldwide.

The objective of this project is to develop a Machine Learning based prediction system that analyzes patient health parameters and predicts whether the patient has a higher risk of heart disease.
